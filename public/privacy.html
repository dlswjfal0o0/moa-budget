개인정보 처리방침

모아(moa) · 최종 수정일: 2026년 7월 15일

모아(이하 "앱")는 사용자의 개인정보를 소중히 여기며, 「개인정보 보호법」 및 관련 법령을 준수합니다. 본 방침은 앱이 수집하는 정보와 그 이용 방식을 설명합니다.

1. 수집하는 개인정보

항목 수집 목적 보관 기간
이메일 주소 회원 가입 및 로그인 식별 회원 탈퇴 시까지
사용자 ID (Firebase UID) 서비스 제공 및 데이터 연동 회원 탈퇴 시까지
가계부 데이터 (거래 내역, 예산, 카테고리) 가계부·예산 관리 서비스 제공 사용자 삭제 또는 탈퇴 시까지
카드 정보 (카드명, 카드번호 끝 4자리, 유효기간, 실적 목표, 결제일) 카드별 지출·실적 관리 서비스 제공 사용자 삭제 또는 탈퇴 시까지
계좌 정보 (계좌명, 잔액, 계좌번호) 계좌별 잔액 관리 서비스 제공 사용자 삭제 또는 탈퇴 시까지
대출 정보 대출 상환 관리 서비스 제공 사용자 삭제 또는 탈퇴 시까지
프로필 사진 프로필 표시 사용자 삭제 또는 탈퇴 시까지

2. 개인정보 수집 방법

  • 회원 가입 시 이메일 주소를 직접 입력
  • 앱 사용 중 사용자가 직접 입력하는 가계부·예산·카드·계좌·대출 데이터 및 프로필 사진

3. 개인정보의 제3자 제공

앱은 원칙적으로 사용자의 개인정보를 외부에 제공하지 않습니다. 단, 아래의 경우에 한해 예외가 있습니다.

  • 법령에 의거하거나 수사 목적으로 관계 기관의 요청이 있는 경우

4. AI 분석 기능 및 개인정보의 국외 이전

분석 기능은 사용자가 직접 입력한 소비 데이터 및 공과금 금액을 기반으로 작동합니다.
분석 처리는 미국 소재 Anthropic, PBC(https://www.anthropic.com)의 API 서버로 해당 데이터를 전송하여 이루어집니다.
이름, 이메일, 카드번호, 계좌번호 등 개인 식별·금융 정보는 AI 서버로 전송되지 않으며, 집계된 카테고리·금액 정보만 전송됩니다.
AI가 생성한 분석 결과는 참고용이며 정확성을 보장하지 않습니다.
이전받는 자 이전 국가 이전 항목 이전 목적 보유·이용 기간
Anthropic, PBC 미국 소비 카테고리·금액 등 집계된 가계부 데이터(개인 식별 정보 제외) AI 소비 분석·조언 제공 Anthropic의 개인정보처리방침 및 API 서비스 약관에 따름 (anthropic.com/legal/privacy)

5. 개인정보 보호 조치

  • 모든 데이터는 Google Firebase를 통해 암호화된 상태로 저장됩니다.
  • 앱과 서버 간 통신은 HTTPS로 암호화됩니다.
  • 사용자 데이터는 해당 사용자 계정에만 접근 가능하도록 Firebase 보안 규칙으로 제한됩니다.
  • 계좌번호는 앱 화면에 기본적으로 마스킹 처리되어 표시되며, 사용자가 직접 전체 보기를 선택한 경우에만 노출됩니다.

6. 사용자의 권리

사용자는 언제든지 아래 권리를 행사할 수 있습니다.

  • 열람: 자신의 개인정보 처리 현황 열람
  • 정정·삭제: 잘못된 정보의 수정 또는 삭제 요청
  • 탈퇴: MY 페이지에서 계정 삭제 시 모든 데이터 즉시 삭제

7. 개인정보 처리방침 변경

본 방침이 변경될 경우 앱 내 공지 또는 이메일을 통해 사전 안내합니다.
